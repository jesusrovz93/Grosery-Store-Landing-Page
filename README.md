# Grosery-Store-Landing-Page


## Descripción

Proyecto frontend de una aplicación web moderna para una tienda de comestibles y productos de supermercado. El objetivo del proyecto es ofrecer una experiencia visual limpia, rápida y responsive para la presentación de productos y navegación de usuarios.

El proyecto fue desarrollado utilizando tecnologías modernas del ecosistema frontend:

* HTML5
* CSS3
* JavaScript
* Vite

La aplicación está enfocada en:

* Rendimiento rápido
* Diseño moderno
* Arquitectura frontend ligera
* Experiencia de usuario responsive

---

# Vista General

La aplicación incluye:

* Landing page moderna
* Configuración optimizada con Vite

# Instalación

## 1. Clonar el repositorio

```bash
git clone https://github.com/tu-usuario/grocery-store.git
```

---

## 2. Entrar al proyecto

```bash
cd grocery-store
```

---

## 3. Instalar dependencias

```bash
npm install
```

---

# Ejecutar el Proyecto

## Desarrollo

```bash
npm run dev
```

Esto iniciará el servidor local de desarrollo utilizando Vite.

---

## Build de producción

```bash
npm run build
```

Genera la versión optimizada lista para despliegue.

---

## Vista previa de producción

```bash
npm run preview
```

Permite visualizar localmente la build de producción.
